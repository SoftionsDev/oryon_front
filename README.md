# Oryon Front


## Getting Started

To set up the project, follow these simple steps:


1. Install dependencies using npm or yarn:
    ```bash
    # Using npm
    npm install

    # Or using yarn
    yarn install
    ```

2. Run the development environment:
    ```bash
    # Using npm
    npm run dev

    # Or using yarn
    yarn dev
    ```

3. Build the project for production:
    ```bash
    # Using npm
    npm run build

    # Or using yarn
    yarn build
    ```
